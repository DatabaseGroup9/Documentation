Feature: Plot Cities By Book
	As a user
	I want to search cities by entering book title
	so that I have an overview of all the cities mentioned in a certain book

	Scenario: Valid Book Title
	Given The Three Masketeers
	When Some cities found
	Then Plot the cities
	
	Scenario: Book Title Not Found
	Given The Book
	When No book found in the database
	Then Display Error Page Book Not Found - No such book in the collection
	
	Examples:
		| 	Book Title	  	   | 		cities 			|		Page		 	 		|
		| The Three Masketeers | Toledo, Madrid, .. 	| Success Page	- plot cities	|
		| The Book	 		   | Null					| Error Page - Not found 		|
